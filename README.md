# 🦷 Smilux Dental Clinic Website

A modern and responsive dental clinic website built using HTML, CSS, and JavaScript.

This project simulates a real clinic workflow, focusing on user experience, booking interactions, and clean UI design.

---

## 🎯 Project Purpose

I built this project to practice creating a real-world frontend system from scratch.

The main goal was to simulate how a dental clinic website works, especially the booking process, including time slot management, validation, and dynamic UI updates — all without relying on frameworks.

---

## 🚀 Features

* 🌙 Light / Dark Mode Toggle (stored in localStorage)
* 🖼️ Hero Image Slider with smooth animation
* 📅 Smart Appointment Booking System
* ⏰ Dynamic Time Slot Selection with unavailable slots handling
* ✅ Custom Form Validation (Contact + Booking)
* 📱 Fully Responsive Design
* 🔝 Back to Top Button
* 🎨 Modern UI using CSS variables and theming

---

## 🧠 Smart Booking Logic

The booking system was implemented manually to simulate real-world behavior:

* Prevent selecting past dates
* Allow booking only on working days (Sunday → Thursday)
* Disable already booked time slots
* Update booking summary in real-time
* Validate all inputs before submission

---

## ⚙️ Technical Decisions

* Used **Vanilla JavaScript** to deeply understand DOM manipulation and state handling
* Implemented **localStorage** for theme persistence instead of external libraries
* Built the booking system logic manually (time slots + availability simulation)
* Focused on reusable and scalable styling using **CSS variables**
* Avoided frameworks intentionally to strengthen core frontend fundamentals
